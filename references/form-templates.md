# 现场质量表单字段详表（知识参考）

> 本文件是 `qms-form-builder` 技能的参考知识：定义三类现场记录表单的字段、下拉项与适用场景。
> 它不是指导文件（SIP/SOP/CP），而是它们的**下一级执行文件**——现场用来"填数据、做记录"的空白表。
> 实质的检验项目、标准、抽样方案由使用者按图纸 / 客户 / SIP 提供并确认。

---

## 一、表单层级关系

```
SIP / SOP / CP   （指导文件：规定"怎么做、判什么"）   ← 由对应技能生成
      │  下一级
      ▼
检验记录表 / 试验测试报告表 / 点检表   （执行文件：现场"填什么、记什么"）   ← 本技能生成
```

---

## 二、检验记录表（首检 / 巡检 / 终检）

**适用**：现场检验员对产品进行首件、过程巡检、完工终检时记录实测与判定。

**表头信息区（自动带出，可填）**

| 字段 | 说明 / 取值示例 |
|------|----------------|
| 表单名称 | 如「首件检验记录表」 |
| 产品名称/型号 | 如实填写 |
| 零件号/图号 | 如 `GB/T 5783` |
| 批次/订单号 | 如 `B-2026-0714` |
| 检验阶段(首检/巡检/终检) | 三选一 |
| 检验依据(SIP/标准) | 如 `SIP-QI-001` |
| 日期 | 填表日期 |
| 班次 | 早班/中班/晚班 |

**字段列（数据区）**

| 序号 | 检验项目 | 标准要求 | 实测值 | 单位 | 判定(合格/不合格) | 不良现象/备注 | 检验员 |
|------|----------|----------|--------|------|-------------------|---------------|--------|
| 1 | … | … | … | … | 合格/不合格 | … | … |

**下拉项**：`判定` 列 = 合格 / 不合格

**签字栏**：检验员 / 审核 / 日期

**签字栏**：测试人 / 复核人 / 日期

## 五、通用约定

- **编号建议**：质量记录（QR）前缀，如 `QR-QI-<流水>`；企业自有规则以使用者为准。
- **判定术语**：合格/不合格（检验/试验）、正常/异常（点检）。
- **抽样 / 标准来源**：GB/T 2828.1、图纸、客户 SIP——由使用者提供，未提供时表单内标注「★待确认」。
- **输出形态**：Excel(.xlsx) 与 Word(.docx) 空白模板，含表头信息区、字段表头、空白数据行、判定/结果下拉、签字栏、打印设置（横向、适应一页宽）。
- **不代替专业技术判断**：检验项目、标准、抽样方案由使用者按上级指导文件提供并确认。
